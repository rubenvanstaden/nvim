# NeoVIM

* Installation can be found at [slackz](https://github.com/rubenvanstaden/slackz)

* Setup configuration

```shell
# Clean user
./script/clean-user.sh

# Install user
./script/install-user.sh

# Compare changes between local and repo
./script/compare.sh
```
